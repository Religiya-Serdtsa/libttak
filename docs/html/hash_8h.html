Detailed Description

Low-level open-addressing hash map (SoA layout) with SipHash/wyhash.

ttak_map_t is the inner SoA table used by the higher-level map and table APIs. Control bytes follow the Swiss-table convention: EMPTY (0x00), DELETED (0xDE), OCCUPIED (0x0C).
