typedef struct ttak_epoch_gc ttak_epoch_gc_t

Epoch-based garbage collection context.

Manages memory lifecycle using generational epochs. Designed for periodic, non-blocking cleanup where the user triggers the cycle. Traverses the heap tree to identify and free expired blocks without a global stop-the-world pause.

void ttak_epoch_gc_rotate ( ttak_epoch_gc_t *  gc)

Advances the epoch and triggers a non-blocking cleanup pass.

This should be called periodically by the user. It increments the epoch and traverses the heap tree to free blocks associated with expired epochs (based on configured retention or explicit lifetime).
