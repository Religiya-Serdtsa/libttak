1/**
2 * @file abstract.h
3 * @brief Pointer-stable logical memory with segmented backing and scoped maps.
4 *
5 * This API exposes a stable logical handle while allowing the implementation to
6 * manage the physical backing as one or more internal segments.
7 *
8 * Core model:
9 * - The handle returned by ttak_abstract_alloc() is the stable identity.
10 * - Logical bytes may be backed by multiple internal segments.
11 * - Callers can use read/write helpers or temporarily map a logical window.
12 * - A mapped window is contiguous from the caller's perspective even when the
13 * implementation must stage it behind the scenes.
14 */
