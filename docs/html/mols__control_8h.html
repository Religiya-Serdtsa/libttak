Detailed Description

Congestion control helpers built on mutually orthogonal Latin squares (MOLS).

Exposes a single entry point that redistributes load across a 64×64 mesh. Each node's coordinates feed a matched pair of Latin squares whose values are XOR-mixed with the caller's seed so every subsystem observes the same deterministic shuffling without branching on compile-time knobs.
