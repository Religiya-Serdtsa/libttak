bool ttak_sum_proper_divisors_u64 ( uint64_t  n,
uint64_t *  result_out 
)

Calculates the sum of proper divisors for a 64-bit unsigned integer.

This function calculates sigma(n) - n. It uses 128-bit intermediate arithmetic to detect overflow.

Parameters
nThe number.
result_outA pointer to store the sum of proper divisors.
Returns
true if the calculation was successful and fits in a uint64_t. false if the sum of divisors overflows uint64_t. In this case, the value in result_out is undefined.
