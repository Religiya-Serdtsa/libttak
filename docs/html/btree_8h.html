Detailed Description

Generic order-N B-tree (multi-way search tree) interface.

Keys and values are stored as void* with caller-supplied comparison and cleanup callbacks. The tree is not thread-safe by itself; wrap accesses in a ttak_rwlock_t when sharing across threads.
