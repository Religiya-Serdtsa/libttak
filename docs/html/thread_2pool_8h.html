Detailed Description

Managed thread pool built on top of the async task scheduler.

Provides a fixed-size pool of worker threads that drain a sharded task queue. Tasks are routed to shards via a deterministic hash-to-coordinate mapping backed by a Latin-square routing table, reducing lock contention by confining queue operations to a single shard lock rather than a global pool lock. Threads are created lazily and respect the nice value set at init time.
