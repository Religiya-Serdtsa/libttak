Detailed Description

Core I/O types: status codes, buffers, TTL guards, and poll helpers.

Defines the shared ttak_io_status_t result type used across all I/O subsystems, the ttak_io_buffer_t staging wrapper, and the ttak_io_guard_t TTL-bounded file-descriptor guard that prevents use of expired or closed descriptors.

◆ ttak_io_guard_t

typedef struct ttak_io_guard ttak_io_guard_t

TTL-bounded guard wrapping a file descriptor.

All I/O operations check the guard before touching the fd. Once expires_at is exceeded the guard is considered invalid and operations return TTAK_IO_ERR_EXPIRED_GUARD.
