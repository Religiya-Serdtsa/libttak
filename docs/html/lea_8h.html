Detailed Description

LEA (Lightweight Encryption Algorithm) block cipher interface.

LEA is a Korean national standard block cipher (KS X 3246) designed for efficient software implementations on 32-bit platforms. Block size is fixed at 16 bytes; supported key lengths are 128, 192, and 256 bits.

typedef struct ttak_lea_schedule ttak_lea_schedule_t

Pre-expanded LEA round-key schedule.

Populated by ttak_lea_schedule_init(). The rounds field is set to 32 (256-bit), 28 (192-bit), or 24 (128-bit) depending on key length.
