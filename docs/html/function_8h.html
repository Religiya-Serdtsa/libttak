Detailed Description

Convenience wrappers for fire-and-forget async jobs that return futures.

Higher level helpers to simplify the pattern of creating a promise/future pair, scheduling a worker task, and awaiting the result.

On any failure, cleanup is invoked with ctx before returning an empty handle to ensure resources aren't leaked.
