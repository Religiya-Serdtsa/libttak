Detailed Description

ROCm/HIP kernel dispatch for arbitrary-precision integer operations.

Mirrors the OpenCL path but uses HIP kernels for AMD GPU targets. On platforms without ROCm the file compiles against the CPU stub headers generated by the CI build script.
