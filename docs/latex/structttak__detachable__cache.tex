\doxysubsection{Detailed Description}
Per-\/context cache for tiny detachable chunks (\texorpdfstring{$<$}{<}= 16 bytes). 

The cache implements an approximated LRU queue that is biased to favor the active generation. Entries are zeroed before returning to the caller to keep calloc semantics intact.
