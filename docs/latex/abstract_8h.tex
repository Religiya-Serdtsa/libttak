\doxysubsection{Detailed Description}
Pointer-\/stable logical memory with segmented backing and scoped maps. 

This API exposes a stable logical handle while allowing the implementation to manage the physical backing as one or more internal segments.

Core model\+:
\begin{DoxyItemize}
\item The handle returned by \doxylink{abstract_8h_a73447bba9538fd5ed9a9d8caf99a1324}{ttak\+\_\+abstract\+\_\+alloc()} is the stable identity.
\item Logical bytes may be backed by multiple internal segments.
\item Callers can use read/write helpers or temporarily map a logical window.
\item A mapped window is contiguous from the caller\textquotesingle{}s perspective even when the implementation must stage it behind the scenes. 
\end{DoxyItemize}
