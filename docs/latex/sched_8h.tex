\doxysubsection{Detailed Description}
Async task scheduler — init, shutdown, schedule, and yield. 

Manages the global worker-\/thread pool. Tasks submitted via \doxylink{sched_8h_ae9db49a4788b2930a8d34216adbe2125}{ttak\+\_\+async\+\_\+schedule()} are picked up by threads in priority order. \doxylink{sched_8h_af9ea1811867a03dbfda35683aeb7d997}{ttak\+\_\+async\+\_\+yield()} relinquishes the CPU slice without sleeping.
