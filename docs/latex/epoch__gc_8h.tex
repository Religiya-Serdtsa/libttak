Epoch-\/based garbage collection context. 

Manages memory lifecycle using generational epochs. Designed for periodic, non-\/blocking cleanup where the user triggers the cycle. Traverses the heap tree to identify and free expired blocks without a global stop-\/the-\/world pause.
