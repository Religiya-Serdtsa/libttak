\doxysubsection{Detailed Description}
Power-\/of-\/two buddy allocator for embedded and bare-\/metal targets. 

Manages a contiguous memory pool using the classic buddy-\/system algorithm. When {\ttfamily EMBEDDED} is defined to 1 the allocator operates without any OS memory services. Three placement policies are available\+: best-\/fit, worst-\/fit, and first-\/fit.
