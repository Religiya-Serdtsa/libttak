\begin{DoxyEnumerate}
\item {\bfseries{Ownership inheritance\+:}} only one side may own locks or free memory at a time. If you skip the lock discipline, shared memory corrupts instantly.
\item {\bfseries{Macro contracts\+:}} macros prefixed with {\ttfamily \+\_\+\+\_\+\+TTAK\+\_\+\texorpdfstring{$\ast$}{*}\+\_\+\+\_\+} flip behavior globally. Failing to reset them can break every module.
\item {\bfseries{Bridge lock\+:}} the context bridge forces all calls through a single mutex so the wrong owner never runs {\ttfamily free}. Modifying it casually defeats the safety guarantee.
\end{DoxyEnumerate}\hypertarget{README.md_autotoc_md210}{}\doxysubsection{\texorpdfstring{Tutorial Outline}{Tutorial Outline}}\label{README.md_autotoc_md210}
