\hypertarget{README.md_autotoc_md129}{}\doxysubsection{\texorpdfstring{Why \$\+O(1)\$?}{Why \$\+O(1)\$?}}\label{README.md_autotoc_md129}
In complex systems with hundreds of owners, iterating through a list to check permissions is slow. By using a bitmap indexed by the Owner\textquotesingle{}s ID, we can check access permissions with a single bitwise operation, regardless of the number of registered owners.
