\doxysubsection{Detailed Description}
Tracks the detach lifecycle of a detachable block. 

The status is stored as a single byte and converges towards UNKNOWN whenever no explicit KNOWN flag is set. Padding keeps 8-\/byte alignment stable for the struct returned to callers.
