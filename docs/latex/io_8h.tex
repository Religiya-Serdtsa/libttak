\doxysubsection{Detailed Description}
Core I/O types\+: status codes, buffers, TTL guards, and poll helpers. 

Defines the shared {\ttfamily \doxylink{io_8h_aae21536ea73a59f5c3f8864a321297a4}{ttak\+\_\+io\+\_\+status\+\_\+t}} result type used across all I/O subsystems, the {\ttfamily \doxylink{io_8h_a13577555a348473d37e420b72805a261}{ttak\+\_\+io\+\_\+buffer\+\_\+t}} staging wrapper, and the {\ttfamily \doxylink{io_8h_aca20f4767510d423ee88de501eb72575}{ttak\+\_\+io\+\_\+guard\+\_\+t}} TTL-\/bounded file-\/descriptor guard that prevents use of expired or closed descriptors.
