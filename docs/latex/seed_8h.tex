\doxysubsection{Detailed Description}
SEED block cipher interface (KS X 1213, ISO/\+IEC 18033-\/3). 

SEED is a 128-\/bit block cipher standardised by the Korean government. Key expansion is stored in {\ttfamily \doxylink{structttak__crypto__ctx_aaa498706513ec8eca10d1fd1396a734e}{ttak\+\_\+crypto\+\_\+ctx\+\_\+t\+::hw\+\_\+state\+::seed}} and the single entry point below performs aligned software encryption.
