{\ttfamily lesson26\+\_\+factorization.\+c} factors 360 as a sanity check; reuse it (and extend it) to validate your routines.\hypertarget{README.md_autotoc_md111}{}\doxysubsection{\texorpdfstring{Engine Notes}{Engine Notes}}\label{README.md_autotoc_md111}
The production engine now mirrors the EPR design goals\+:


\begin{DoxyEnumerate}
\item {\bfseries{Trial Division Wheel}} – Quickly strips the first 100 primes (including 2) while respecting the library allocator contracts.
\item {\bfseries{Deterministic Miller-\/\+Rabin (64-\/bit)}} – Bases \{2,3,5,7,11,13\} avoid undefined behavior and allow us to skip ECM for most workloads.
\item {\bfseries{Pollard-\/\+Rho with Brent Switching}} – Pure C, lock-\/free randomization seeded from the caller’s timestamp. It gracefully degrades into deterministic trial division if a stubborn composite survives 32 attempts.
\item {\bfseries{Bigint Fast Path}} – When a bigint fits inside 64 bits we reuse the optimized {\ttfamily \doxylink{factor_8h_ae6b18eac3f9615ae93f70bbe09218716}{ttak\+\_\+factor\+\_\+u64()}} output to avoid repeated allocations.
\end{DoxyEnumerate}
