\doxysubsection{Detailed Description}
Minimal SHA-\/256 hash interface. 

Provides a standard init/update/final streaming hash API. The output digest is 32 bytes (256 bits) written to the caller-\/supplied array in ttak\+\_\+sha256\+\_\+final().
