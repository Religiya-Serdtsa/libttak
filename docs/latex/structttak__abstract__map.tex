\doxysubsection{Detailed Description}
Scoped mapping of a logical byte window in an abstract object. 

A map pins relocation behind the object\textquotesingle{}s synchronization boundary and exposes a contiguous byte window for direct indexed access. The mapping is valid only until \doxylink{abstract_8h_aaefa1f99637a1519d420edfd0adf2439}{ttak\+\_\+abstract\+\_\+unmap()} is called.
