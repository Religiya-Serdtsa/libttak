\doxysubsection{Detailed Description}
Architecture-\/aware memory primitives for Tiny\+CC builds. 

Exposes streaming zero/copy helpers that fall back to plain C when the target ISA is not covered or when a non-\/\+Tiny\+CC compiler is driving the build. The helpers retain memcpy/memset semantics (non-\/overlapping copy, byte-\/exact zeroing) but are written so libttak can keep high throughput even when Tiny\+CC emits -\/O0 code.
