\doxysubsection{Detailed Description}
Simplified macros that return a plain pointer for caller-\/driven lifetime. 

These macros wrap the underlying {\ttfamily \+\_\+safe} functions with sensible defaults for everyday use. The caller is responsible for freeing the returned memory.
