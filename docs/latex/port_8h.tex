\doxysubsection{Detailed Description}
Platform-\/agnostic network driver operations and OS detection. 

Defines the {\ttfamily ttak\+\_\+net\+\_\+driver\+\_\+ops\+\_\+t} vtable that abstracts socket calls across POSIX, Windows, and bare-\/metal targets. Call \doxylink{port_8h_a782d7b255df32e840f51eb6230204d56}{ttak\+\_\+net\+\_\+driver\+\_\+detect()} once at startup to populate the vtable.
