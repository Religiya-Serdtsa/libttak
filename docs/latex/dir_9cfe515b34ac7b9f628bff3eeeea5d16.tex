\doxysubsection{Detailed Description}
\hypertarget{README.md_autotoc_md59}{}\doxysubsection{\texorpdfstring{Tutorial 01 – Getting Started}{Tutorial 01 – Getting Started}}\label{README.md_autotoc_md59}
This folder hosts the hands-\/on assets that accompany {\ttfamily modules/01-\/getting-\/started}. Use it after you have already built and {\bfseries{installed}} {\ttfamily libttak} (e.\+g., {\ttfamily make \&\& sudo make install}). The intent is to prove that your toolchain is wired correctly and to watch the lifetime APIs work in a tiny, self-\/contained program.\hypertarget{README.md_autotoc_md60}{}\doxysubsection{\texorpdfstring{What you will do}{What you will do}}\label{README.md_autotoc_md60}

\begin{DoxyItemize}
\item Compile {\ttfamily getting\+\_\+started.\+c}, which allocates a short-\/lived message with {\ttfamily ttak\+\_\+mem\+\_\+alloc}, inspects it midway through its lifetime, and observes expiration.
\item Run the program and compare the log output to the lesson checklist.
\item Jot down your "{}\+What I learned"{} entry before moving on to Lesson 02.
\end{DoxyItemize}\hypertarget{README.md_autotoc_md61}{}\doxysubsection{\texorpdfstring{Building the sample}{Building the sample}}\label{README.md_autotoc_md61}
