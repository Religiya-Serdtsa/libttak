\doxysubsection{Detailed Description}
One-\/shot promise for fulfilling a ttak\+\_\+future\+\_\+t from a worker thread. 

Create a promise with \mbox{\hyperlink{promise_8h_af7cd6c810a6e79977479b5128ad5dddf}{ttak\+\_\+promise\+\_\+create()}}, obtain its future with \mbox{\hyperlink{promise_8h_a6d7bfc3b57e932e249ffc0f43c0fd6c9}{ttak\+\_\+promise\+\_\+get\+\_\+future()}}, and fulfil it from the worker thread via \mbox{\hyperlink{promise_8h_ae4d7300d418736b6deb56f8c19d8b52f}{ttak\+\_\+promise\+\_\+set\+\_\+value()}}.
