\doxysubsection{Detailed Description}
Number-\/\+Theoretic Transform (NTT) and Montgomery arithmetic. 

Provides NTT-\/based polynomial multiplication over prime fields for big-\/integer arithmetic. Three built-\/in NTT primes are included, all of the form p = c $\ast$ 2$^\wedge$k + 1, supporting transforms up to 2$^\wedge$max\+\_\+power\+\_\+two elements. Chinese Remainder Theorem (CRT) reconstruction is provided to lift results back to a larger ring.
