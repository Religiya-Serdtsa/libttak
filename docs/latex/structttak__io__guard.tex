\doxysubsection{Detailed Description}
TTL-\/bounded guard wrapping a file descriptor. 

All I/O operations check the guard before touching the fd. Once {\ttfamily expires\+\_\+at} is exceeded the guard is considered invalid and operations return {\ttfamily TTAK\+\_\+\+IO\+\_\+\+ERR\+\_\+\+EXPIRED\+\_\+\+GUARD}.
