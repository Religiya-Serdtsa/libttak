\doxysubsection{Detailed Description}
Low-\/level open-\/addressing hash map (SoA layout) with Sip\+Hash/wyhash. 

{\ttfamily \doxylink{structttak__map__t}{ttak\+\_\+map\+\_\+t}} is the inner SoA table used by the higher-\/level map and table APIs. Control bytes follow the Swiss-\/table convention\+: {\ttfamily EMPTY} (0x00), {\ttfamily DELETED} (0x\+DE), {\ttfamily OCCUPIED} (0x0C).
