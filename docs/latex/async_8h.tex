\doxysubsection{Detailed Description}
Non-\/blocking I/O read/write operations with completion callbacks. 

Wraps the platform poll/select mechanism to schedule asynchronous reads and writes on a guarded file descriptor. The callback is invoked on the worker thread when the operation completes or times out.
