 This directory houses the Plant\+UML ({\ttfamily .puml}) blueprints for {\ttfamily libttak}. Each file now lives in a role-\/oriented folder so it is easier to navigate the architecture layers.\hypertarget{index_autotoc_md1}{}\doxysection{\texorpdfstring{Directory Structure}{Directory Structure}}\label{index_autotoc_md1}

\begin{DoxyItemize}
\item {\ttfamily application/}\+: Product-\/entry diagrams such as {\ttfamily \doxylink{00__overview_8puml}{00\+\_\+overview.\+puml}}.
\item {\ttfamily logic/}\+: Specialized logic (math kernels, tree structures).
\item {\ttfamily execution/}\+: Runtime and scheduling blueprints.
\item {\ttfamily resource/}\+: Ownership, epoch GC, and detachable/arena allocators.
\item {\ttfamily foundation/}\+: Threading, synchronization, and utility services.
\item {\ttfamily io\+\_\+net/}\+: Guarded I/O, buffers, and network topology.
\item {\ttfamily embedded/}\+: Bare-\/metal specific allocators and drivers.
\item {\ttfamily integration/}\+: Cross-\/cutting diagrams like {\ttfamily \doxylink{all_8puml}{all.\+puml}}.
\item {\ttfamily png/}\+: Rendered PNG files that mirror the same folder hierarchy.
\item {\ttfamily scripts/}\+: Rendering helpers (currently {\ttfamily render\+\_\+blueprints.\+py}).
\end{DoxyItemize}\hypertarget{index_autotoc_md2}{}\doxysubsection{\texorpdfstring{Notable Blueprints}{Notable Blueprints}}\label{index_autotoc_md2}

To update the PNG files from the {\ttfamily .puml} sources, run the following command from the project root\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{make\ blueprints}

\end{DoxyCode}


Or run the script directly\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{python3\ blueprints/scripts/render\_blueprints.py}

\end{DoxyCode}


The script will automatically check for changes and only re-\/render files that have been modified. It attempts to use a local {\ttfamily plantuml} installation first, and falls back to the Plant\+UML online server if necessary.\hypertarget{index_autotoc_md4}{}\doxysection{\texorpdfstring{Adding New Blueprints}{Adding New Blueprints}}\label{index_autotoc_md4}

\begin{DoxyEnumerate}
\item Create a new {\ttfamily .puml} file in the folder that matches the role (or create a new folder if a new role emerges).
\item Run {\ttfamily make blueprints}.
\item The resulting PNG will be stored in the mirrored path under {\ttfamily blueprints/png/}. 
\end{DoxyEnumerate}
