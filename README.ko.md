# LibTTAK (한국어 안내)

LibTTAK은 C 코드에서 메모리 수명 관리를 명시적으로 다루기 위한 시스템 라이브러리입니다.

- Arena/epoch 기반 메모리 수명 관리
- lock-free 구조 및 스케줄링 구성요소
- 벤치마크/튜토리얼/문서 포함

기본 영문 문서는 [`README.md`](./README.md)를 참고하세요.
