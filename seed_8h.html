Detailed Description

SEED block cipher interface (KS X 1213, ISO/IEC 18033-3).

SEED is a 128-bit block cipher standardised by the Korean government. Key expansion is stored in ttak_crypto_ctx_t::hw_state::seed and the single entry point below performs aligned software encryption.

Encrypts data using SEED with a 16-byte-aligned software path.

All parameters (plaintext, output buffer, pre-expanded round keys) are taken from ctx. Each caller must own a separate context instance.
